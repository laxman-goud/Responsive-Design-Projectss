Introduction to CSS

CSS stands for Cascading Style Sheets and is used to style HTML content.

It controls the layout, colors, fonts, and overall presentation of web pages.

body { background-color: #f0f0f0; }
  • CSS adds visual appeal to websites.
  • CSS can be internal, external, or inline.
  • CSS improves user experience.
Selectors and Properties

Selectors target HTML elements, and properties apply styles to them.

Common selectors include element, class, ID, and attribute selectors.

p { font-size: 16px; }
  • Element Selector: targets all elements of a type.
  • Class Selector: targets elements with a specific class.
  • ID Selector: targets a unique element with an ID.
Box Model

The CSS box model describes how elements are rendered in terms of space.

It includes margins, borders, padding, and content.

div { margin: 10px; padding: 15px; }
Positioning and Layout

CSS provides different positioning methods: static, relative, absolute, and fixed.

Layout models include Flexbox and Grid for responsive alignment.

.container { display: flex; justify-content: center; }
Responsive Design

Responsive design ensures web content adapts to various screen sizes.

Media queries allow CSS to apply styles conditionally.

@media (max-width: 600px) { body { font-size: 14px; } }
